# microservice-nestJS

### 1. Запустить базы данных
 ```
 cd docker

 docker-compose up -d
 ```

### 2. Запустить сервис account
  ```
  cd account

  yarn start
  ```

### 3. Запустить сервис auth
  ```
  cd auth

  yarn start
  ```

### 4. Запустить сервис transaction
  ```
  cd transaction

  yarn start
  ```
